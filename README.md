>This is a simple example of Spring Boot Kafka
>

## Prerequisites

*  Docker
*  Docker Compose

## Installation

```sh
docker-compose up -d
```

## Used Dependencies
* Spring Boot Web
* Spring Kafka
* Spring Boot Actuator
* Lombok
* Configuration Processor
